## MCQ 1
Question: What is Git?
A) A graphical user interface for interacting with repositories.
B) A distributed version control system.
C) A platform-specific installer.
D) A website for hosting code.
Correct Answer: B

## MCQ 2
Question: Which command displays modified files in the working directory staged for the next commit?
A) `git add`
B) `git status`
C) `git commit`
D) `git diff`
Correct Answer: B

## MCQ 3
Question: How do you add a file to the next commit?
A) `git commit [file]`
B) `git add [file]`
C) `git status [file]`
D) `git reset [file]`
Correct Answer: B

## MCQ 4
Question: What does `git config --global user.name “[firstname lastname]”` do?
A) Sets the user's email address.
B) Sets the user's name for version history.
C) Sets the global color scheme.
D) Initializes a Git repository.
Correct Answer: B

## MCQ 5
Question: Which command initializes an existing directory as a Git repository?
A) `git clone`
B) `git init`
C) `git config`
D) `git remote`
Correct Answer: B

## MCQ 6
Question:  What does `git log branchB..branchA` display?
A) Commits on both branchA and branchB.
B) Commits on branchA that are not on branchB.
C) Commits on branchB that are not on branchA.
D) All commits on the current branch.
Correct Answer: B

## MCQ 7
Question: Which command deletes a file from the project and stages the removal for commit?
A) `git mv`
B) `git rm`
C) `git log`
D) `git reset`
Correct Answer: B

## MCQ 8
Question:  What is the purpose of `.gitignore`?
A) To track file changes.
B) To specify files to be excluded from staging or committing.
C) To rewrite branch history.
D) To store temporary commits.
Correct Answer: B

## MCQ 9
Question: What does `git stash` do?
A) Discards changes from the stash stack.
B) Saves modified and staged changes temporarily.
C) Lists the stash stack order.
D) Applies stashed changes to the working directory.
Correct Answer: B

## MCQ 10
Question: What is the command to retrieve an entire repository from a hosted location via URL?
A)  `git init [url]`
B)  `git clone [url]`
C)  `git pull [url]`
D)  `git fetch [url]`
Correct Answer: B
