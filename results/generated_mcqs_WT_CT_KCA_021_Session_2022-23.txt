## MCQ 1
**Question:** What is the purpose of the `<!DOCTYPE>` tag in HTML?
A) It defines the document type and version of HTML being used.
B) It specifies the character encoding for the document.
C) It links the HTML document to an external stylesheet.
D) It defines a new HTML element for use in the document.
**Correct Answer:** A) It defines the document type and version of HTML being used.

## MCQ 2
**Question:**  Which of the following is NOT a valid data type in JavaScript?
A) String
B) Number
C) Boolean
D) Character
**Correct Answer:** D) Character

## MCQ 3
**Question:**  What is the role of a Servlet in web development?
A) It handles client-side scripting and user interactions.
B) It stores and retrieves data from a database.
C) It acts as a server-side component to process client requests and generate dynamic content.
D) It defines the structure and layout of a web page.
**Correct Answer:** C) It acts as a server-side component to process client requests and generate dynamic content.

## MCQ 4
**Question:** Which CSS property is used to add space around the content inside an element, while the margin property adds space around the element itself?
A) Border
B) Padding
C) Outline
D) Spacing
**Correct Answer:** B) Padding

## MCQ 5
**Question:** What is the difference between an ordered list and an unordered list in HTML?
A) Ordered lists use roman numerals while unordered lists use numbers.
B) Ordered lists display items in a specific sequence, while unordered lists display items without any order. 
C) Ordered lists are styled with bullets, while unordered lists are styled with numbers or letters.
D) There is no difference, both are interchangeable.
**Correct Answer:** B) Ordered lists display items in a specific sequence, while unordered lists display items without any order. 

## MCQ 6
**Question:** What is the primary purpose of the JavaScript `forward()` method in a Servlet?
A) To redirect the user to a different website.
B) To forward the request to another resource on the server for further processing.
C) To send an email notification to the client.
D) To display an error message to the user.
**Correct Answer:** B) To forward the request to another resource on the server for further processing.

## MCQ 7
**Question:**  According to Bloom's Taxonomy framework used in the text, what does the abbreviation "KC" stand for?
A) Knowledge Competency 
B) Key Concept
C) Knowledge Categories
D) Knowledge Comprehension
**Correct Answer:** C) Knowledge Categories
